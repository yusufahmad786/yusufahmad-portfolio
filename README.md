# Yusuf Ahmad | Personal Website (One Page)

A lightning fast, mobile first one page portfolio built with **React + Vite + Tailwind CSS**.

Includes:
- Canvas animated background with software testing skills
- Storybook for UI components
- Jest for unit tests
- Cypress for end to end tests
- Vercel ready (SPA rewrite included)

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Storybook

```bash
npm run storybook
```

## Unit tests (Jest)

```bash
npm test
```

## E2E tests (Cypress)

Run the dev server first, then:

```bash
npm run cypress:open
```

## Deploy to Vercel

1) Push this repo to GitHub
2) Import it in Vercel
3) Framework preset: Vite
4) Build command: `npm run build`
5) Output directory: `dist`

## Update content

Edit data here:
- `src/data/profile.js`

Assets:
- `public/assets` (logos and previews)
- `public/docs` (resume and certificates)
